# PDF Редактор

Простое веб-приложение для работы с PDF файлами.

## Функции

- Загрузка PDF файлов
- Навигация по страницам
- Масштабирование (приближение и отдаление)
- Печать
- Сохранение страниц в виде изображения

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- PDF.js для отрисовки PDF
- Hammer.js для гестов

## Как использовать

1. Откройте `index.html` в браузере
2. Нажмите "Открыть PDF"
3. Выберите PDF файл ис вашего компьютера
4. Используйте кнопки для навигации и редактирования
